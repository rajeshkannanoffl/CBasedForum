## C - Program Notes

```
Standard variable stores value.
Pointer stores address of the variable.
----------------------------------------
Pointer Initialization: Initialize pointer to point the address of variable.
1. *p -> throws garbage value.
2. *p = NULL -> defined.
3. *p = &a -> ampersand to get the address of var a.
-----------------------------------------
Pointer Deferencing: Assigning value for variable through pointer.
1. * -> Indirection pointer
2. * -> used with pointer to assign value.
------------------------------------------
Pass By Reference: 
1. Passing address of the variable instead of passing a value & accessing it using pointer.
------------------------------------------
Summary:
1. Operator & - obtains the address of the variable.
2. Pointer Variable - points to the address of the value.
3. Operator * - used for pointer variable, access of content of address of variable.
```
